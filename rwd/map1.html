HTML <map> Tag and <area> Tag

The <map> and <area> tags are used together to create an image map. An image map is an image with clickable areas.

1. <map> Tag

The <map> tag defines an image map. It does not display anything itself, but it is linked to an image using the usemap attribute of the <img> tag.

Important attributes of <map>:

  • name – a unique name used to link the map with an image.

2. <area> Tag

The <area> tag defines each clickable area inside the map. It is always written inside the <map> tag.

Important attributes of <area>:

  • shape – shape of the area (rect, circle, poly).
  • coords – coordinates of the area (depends on shape).
  • href – link to open when area is clicked.
  • alt – alternative text (for accessibility).
  • title – text shown as tooltip on hover (optional but useful).

3. Simple Image Map Example

In this example, we have an image with two clickable areas:

  • A rectangular area on the left side that opens Google.
  • A circular area on the right side that opens YouTube.

4. HTML Code for the Example

You can use this code to create your own image map:

<img src="https://th.bing.com/th/id/OIP.cjfGiqW-cFpC7vsx1dxFAgHaEK?w=276&h=180&c=7&r=0&o=7&pid=1.7&rm=3"
     alt="Sample Image Map"
     usemap="#sampleMap">

<map name="sampleMap">
    <!-- Rectangle clickable area -->
    <area shape="rect"
          coords="10,10,290,190"
          href="https://www.google.com"
          alt="Go to Google"
          title="Open Google">

    <!-- Circle clickable area -->
    <area shape="circle"
          coords="450,100,80"
          href="https://www.youtube.com"
          alt="Go to YouTube"
          title="Open YouTube">
</map>

5. Summary

  • <map> defines the image map and groups the areas.
  • <area> defines each clickable region on the image.
  • usemap in <img> connects the image to the map.
